# WKT Spatial Editor

A modern web application for editing and visualizing Well-Known Text (WKT) geometries with an interactive spatial editor.

## 🚀 Live Deployment

View the live application: **[https://wkt-spacial-editor.netlify.app/](https://wkt-spacial-editor.netlify.app/)**

## 📋 Tech Stack

- **React** - UI framework
- **TypeScript** - Type-safe development
- **Vite** - Fast build tool with HMR
- **Tailwind CSS** - Styling
